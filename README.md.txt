Controle de Metas Financeiras

Projeto em Java para gerenciamento de metas financeiras pessoais via terminal. Criado com foco em aprendizado, prática de orientação a objetos e tratamento de erros.

---

🎯 Funcionalidades

- ✅ Cadastro de metas com nome, valor total e valor já poupado
- ✅ Cálculo automático do progresso em porcentagem
- ✅ Validação de entradas com tratamento de erros
- ✅ Organização do código em classes separadas (POO)
